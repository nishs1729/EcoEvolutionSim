# 1. Agent Representation

Each insect agent has a phenotype vector:

```
z = (
    z1  resource_preference
    z2  niche_width
    z3  life_history_speed
    z4  mating_investment
    z5  mate_control
    z6  defense_investment
    z7  dispersal_ability
    z8  behavioral_plasticity
)
```

Additional dynamic state variables:

```
energy
age
location
sex
stored_sperm
```

The environment contains:

```
resource_field
predator_field
mate_locations
```

# 2. Resource Acquisition Rules

Resource types exist along a gradient (r).

Feeding efficiency depends on the resource trait axes.

Resource intake function:

$$E(r) = \exp\left(-\frac{(r-z_1)^2}{2z_2^2}\right)$$

where

* ($z_1$) = preferred resource
* ($z_2$) = niche width

Energy intake:

```
energy_gain = resource_density * E(r)
```

Consequences:

* small ($z_2$) → specialist
* large ($z_2$) → generalist

# 3. Movement / Dispersal Rules

Axis ($z_7$) controls mobility.

Movement step size:

```
step_size = base_step * (1 + z7)
```

Long-distance dispersers therefore explore more habitat.

Plasticity ($z_8$) modifies movement decisions:

```
if plasticity high:
    move toward highest resource patch
else:
    random walk
```

---

# 4. Defense Against Predators

Axis ($z_6$) determines predator survival probability.

Example rule:

```
predation_risk = base_predation * exp(-z6)
```

High defense reduces mortality.

But defense also costs energy:

```
energy_cost += defense_cost * z6
```

Tradeoff:

```
defense ↔ reproduction
```

---

# 5. Life History Rules

Axis ($z_3$) controls the **speed of life history**.

Example mappings:

```
development_time = base_dev_time / (1 + z3)

fecundity = base_fecundity * (1 + z3)

lifespan = base_lifespan / (1 + z3)
```

So:

high ($z_3$):

* fast reproduction
* short life

low ($z_3$):

* slow reproduction
* long life

This reproduces the **r–K continuum**.

# 6. Mate Encounter Behavior

Axis ($z_4$) determines **investment in mate attraction**.

Signal strength:

```
signal_strength = z4 * energy
```

Mate attraction radius:

```
attraction_radius = base_radius * (1 + z4)
```

High values produce behaviors like:

* pheromone emission
* courtship display

Low values produce:

* scramble competition

---

# 7. Mate Control / Sexual Conflict

Axis ($z_5$) determines coercive behavior.

Mating probability:

```
P_mate = female_acceptance + z5 * coercion_factor
```

If ($z_5$) is high:

* forced mating becomes possible

If low:

* mating depends entirely on female choice

---

# 8. Behavioral Plasticity

Axis (z_8) determines **adaptive decision-making**.

Low plasticity:

```
behavior fixed
```

High plasticity:

```
choose action maximizing expected energy gain
```

Example decision:

```
choose between:
    search for food
    search for mate
```

based on environment.

---

# 9. Energy Budget

Every action changes energy.

Energy flow example:

```
energy += feeding_gain
energy -= movement_cost
energy -= defense_cost
energy -= signaling_cost
```

Energy determines survival and reproduction.

---

# 10. Reproduction

Egg production depends on:

```
offspring = fecundity * energy
```

Trait inheritance:

```
z_offspring = z_parent + Normal(0, mutation_variance)
```

This causes gradual evolutionary change.

---

# 11. Survival Probability

Survival is determined by:

```
P_survival =
    energy_factor
    * defense_factor
    * predation_risk
    * environment_tolerance
```

Individuals with better trait combinations survive.

# 13. Emergent Speciation

When this model includes **spatial structure**, something interesting happens.

If resource types vary spatially:

```
resource gradient → divergent selection
```

This produces:

```
different z1 values in different regions
```

If mating preferences evolve simultaneously:

```
assortative mating by phenotype
```

then populations split.

This produces **ecological speciation**.

# 14. Overall Model Flow

Simulation loop:

```
for each timestep:

    grow resources

    for each insect:

        move()
        feed()
        avoid predators()
        search_for_mates()
        reproduce()

    apply mortality
    mutate traits
```
